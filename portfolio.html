Web Design

  • 15637 Social Network

    A Twitter-like social network built by Django, achieved features including user registration and authentication, input validation, photo uploading, quasi-real-time updates, and following system. Finished both front and back ends development individually.

  • OBATTLE

    OBATTLE is a HTML5 web game. I made it when I attended the IDEA-X Summer School in Centre For Digital Media in Canada. I learnt HTML5 there, and made this game with my friends.

  • Guang Yin

    Guangyin is a website introducing Chinese traditional Twenty-four Solar Terms. It is an initial project of my javascript study. I made this website with one partner.

  • NZ South Island

    I created this website in memory of my amazing travel to New Zealand South Island. I finished it with a friend who helped me collect tourism information. I designed the layout and wrote most of the code.

  • Wang Jiang Hotel

    This is a demo website for managing a hotel, including an online booking system for customers, and a background management system for administrators. I bulit this website for practicing my PHP & MySQL study.

Game Development

  • Kungfu Monk

    A VR game works on HTC VIVE, finished in only two weeks on BVW class, by a small team of 5 people including me as a programmer. Players need to pass a series of challenges by using swords to hit and dodge incoming darts, to become a kungfu master.

  • Leap Music

    A 3D music game played on Leap Motion. Players use finger gestures like swing to match rhythm. This game is made by Unity, and there are two versions for both PC and Mac.

  • Circles (iOS)

    Circles is an iOS game, the rule is same as OBATTLE, absorb the smaller and avoid being absorbed by the bigger. We redesigned the UI, to ensure it fits mobile devices. I made this game by Adobe Air, and it works well on both iPhone and iPad.

  • Circles (Android)

    This is Circles Android version. We redesigned the UI and music again, now it looks more beautiful and professional. It is still made by Adobe Air, adapting the differenet resolustions on every Android device automatically.

  • Apocalypse Tank

    This is a 3D model of Apocalypse Tank. I made it by 3ds Max.

  • GAUSS Calculator

    GAUSS Calculator is an iPhone calculator app, which can set varibles, check formulas and review calculator history. I made this app when I learnt Xcode and Objective-C to creat iOS apps.
